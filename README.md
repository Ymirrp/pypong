# PyPong

Run main.py for single-player
```bash
$ python main.py
```

or for multi-player
```bash
$ python main.py 2p
```

# Controls
## Single-Player
<kbd>Up</kbd> and <kbd>Down</kbd> or using the mouse (only available in single-player mode).

## Multi-Player
### 1P
<kbd>W</kbd> and <kbd>S</kbd>
### 2P
<kbd>Up</kbd> and <kbd>Down</kbd>
